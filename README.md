goahead2.5的源代码，做了少量修改

---

**作用：**  

* 便于直接调试页面代码，看到页面效果，模拟后台数据更加方便。  

---

**修改点：**  

* 重写goform的回调函数，在回调函数里把PATH改为请求url后缀.txt的形式，然后返回0，使其进入默认处理动作，返回对应的.txt的json数据文件  

* 添加webs.conf文件，初始化时读取这个文件的配置进行初始化，增加易用性  

---

**编译：**    
进入LINUX目录，make  
  
**使用：**  
配置LINUX/webs.conf文件，运行./webs
